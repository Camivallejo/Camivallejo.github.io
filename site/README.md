# Portfolio — Camila Vallejo Calizaya

Sitio de portafolio (HTML estático, sin dependencias de build).

## Estructura
```
index.html              · la página
support.js              · runtime necesario (no borrar)
l1-3--mqiuzpox.png      · retrato del hero
about-1/2/3-*.jpg       · collage de "Sobre mí"
images/projects/        · portadas y galerías de cada proyecto
```

## Publicar en GitHub Pages
1. Crea un repositorio nuevo en GitHub y sube **todo el contenido de esta carpeta** (que `index.html` quede en la raíz del repo).
2. En el repo: **Settings → Pages**.
3. En *Build and deployment* elige **Deploy from a branch**, rama `main`, carpeta `/ (root)`. Guarda.
4. En ~1 min tu sitio estará en `https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`.

## Notas
- Las tipografías (Cormorant Garamond, Hanken Grotesk, IBM Plex Mono) se cargan desde Google Fonts, así que se necesita conexión a internet.
- Para reemplazar el retrato o las fotos de "Sobre mí", sustituye los archivos manteniendo el mismo nombre.
- Para cambiar imágenes de proyectos, reemplaza los archivos dentro de `images/projects/` con el mismo nombre (`<proyecto>-cover.jpg`, `<proyecto>-1.jpg`, etc.).
